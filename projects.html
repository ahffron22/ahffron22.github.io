Entry Level Projects

Trivia Game
 This is a simple trivia game that gives you thirty seconds to answer three trivia questions. It will tell you everytime you select an answer if it is correct or not.
Train Schedule
 This app allows you to add trains to the train schedule and figures out when the next one will be arriving. It also saves all entries to Firebase and populates the table with all new entries.
Crystal Adding Game
 This app asigns a random number to each crystal each round and generates a random number that you have to try and use the crystal values to reach.

Intermediate Projects

Movie Finder
 This app makes calls to two api databases to retrieve relevant info on any movie you search.
Kitsune Restaurant
 This is one of the first multi functional websites that I built, and was for a friend of mine who started a small catering and food delivery service. This site allows you to add and remove things from your cart, saves order progress with session storage, and allows the admin user to update the specials menu from a hidden page. The specials menu is then saved to a firebase database and populates the menu with up to date specials. It also uses the paypal api to allow users multiple ways to pay.
